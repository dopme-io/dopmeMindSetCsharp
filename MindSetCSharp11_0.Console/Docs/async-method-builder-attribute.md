# `[AsyncMethodBuilder]` em métodos (C# 10.0)

## Contexto

O atributo **`AsyncMethodBuilder`** pode aplicar-se a **métodos** individuais (não só a tipos), permitindo **builders** personalizados por método — usado no **.NET 6** para otimizar caminhos **async** (`pooling`, etc.).
