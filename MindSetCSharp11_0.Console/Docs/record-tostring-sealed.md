# `sealed` em `ToString()` de *record* (C# 10.0)

## Contexto

Num tipo **`record`**, pode marcar **`override string ToString()`** como **`sealed`** para impedir **novas substituições** em tipos derivados — controlo da representação textual em hierarquias.
