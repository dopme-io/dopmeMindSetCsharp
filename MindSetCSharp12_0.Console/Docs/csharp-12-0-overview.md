# C# 12.0 — visão geral (novembro 2023)

## Contexto

**C# 12.0** acompanha **.NET 8** e reforça a **cadência anual**. A ideia geral: **sintaxe que já conhece** (construtores, coleções, `ref`) **em mais sítios**, e **consistência** entre conceitos relacionados. Há também recursos marcados como **experimentais** ou **pré-visualização** (*interceptors*).

O projeto **MindSetCSharp12_0.Console** define **`LangVersion` 12** no `.csproj`.
