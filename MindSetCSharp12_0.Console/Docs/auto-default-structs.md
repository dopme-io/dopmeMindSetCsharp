# *Auto-default structs* (C# 11.0)

## Contexto

O compilador **inicializa campos de struct** para o **default** do tipo quando o construtor não os cobre explicitamente, alinhando melhor o comportamento com a regra de **default** de `struct` e reduzindo atrito em tipos valor.

## O que praticar

Reveja [structure-types-improvements.md](structure-types-improvements.md) (C# 10) e [structs.md](structs.md) (base); experimente structs com campos deixados por omissão no construtor com C# 11.
