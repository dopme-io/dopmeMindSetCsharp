# *Inline arrays* (C# 12.0)

## Contexto

Com **`[System.Runtime.CompilerServices.InlineArray(N)]`** num `struct`, define-se um **buffer de tamanho fixo** `N` embutido no tipo valor — útil para APIs de alto desempenho e interop, com semântica de **array inline** gerida pelo runtime.
