# Declaração de `namespace` ao âmbito do ficheiro (C# 10.0)

## Contexto

```csharp
namespace MyApp.Feature;

class Foo { }  // membro de MyApp.Feature sem chaves extra
```

Uma linha **`namespace X.Y;`** faz com que **todo** o ficheiro pertença a esse namespace — **menos indentação** e leitura mais clara.
