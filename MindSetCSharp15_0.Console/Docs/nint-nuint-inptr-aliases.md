# `nint` / `nuint` como aliases de `IntPtr` / `UIntPtr` (C# 11.0)

## Contexto

Em C# 11, **`nint`** e **`nuint`** são **aliases** linguísticos de **`System.IntPtr`** e **`System.UIntPtr`** (tamanho nativo da plataforma), unificando o modelo com tipos interop e código *unsafe*.

Relacione com [native-sized-integers.md](native-sized-integers.md) (C# 9), onde `nint`/`nuint` foram introduzidos no conjunto da linguagem.
