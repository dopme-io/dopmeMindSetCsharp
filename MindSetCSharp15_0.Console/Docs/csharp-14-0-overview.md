# C# 14.0 — visão geral (novembro 2025)

## Contexto

**C# 14.0** acompanha **.NET 10** e alarga **extensões**, atribuições com **null-condicional**, **`nameof`** em tipos genéricos não ligados, **`Span<T>`** / **`ReadOnlySpan<T>`**, lambdas com **modificadores** nos parâmetros simples, propriedades com **`field`**, tipos **parciais** com eventos e construtores, e **operadores de atribuição composta** definidos pelo utilizador.

O projeto **MindSetCSharp14_0.Console** define **`LangVersion` 14** no `.csproj`.
