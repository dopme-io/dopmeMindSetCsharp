# Propriedades e indexadores parciais (C# 13.0)

## Contexto

Em tipos **`partial`**, **propriedades** e **indexadores** podem ser declarados de forma **parcial** — por exemplo, `get`/`set` ou implementação repartida por ficheiros (*source generators*, hand-written partials).
