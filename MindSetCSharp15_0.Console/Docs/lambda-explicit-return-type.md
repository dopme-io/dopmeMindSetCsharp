# Tipo de retorno explícito em lambda (C# 10.0)

## Contexto

Quando a inferência **não** determina o tipo de retorno, pode declarar **`static`** (C# 9) e, em C# 10, o **tipo de retorno** na assinatura da lambda: `Func<int, string> f = int (x) => x.ToString();` (sintaxe ilustrativa; ver especificação/regras exactas).
