# Atributo `Experimental` (C# 12.0)

## Contexto

**`[Experimental(diagnosticId)]`** marca APIs como **experimentais**; o compilador pode emitir **avisos** quando código de produção as consome, alinhando com evolução controlada de bibliotecas.

## O que praticar

Aplicar o atributo a tipos/membros públicos e observar o **diagnostic ID** nas chamadas.
