# C# 1.0 — Roteiro MindSet

**Versão da linguagem:** 1.0  
**Contexto:** lançada em janeiro de 2002, com o Visual Studio .NET 2002, alinhada aos objetivos ECMA: linguagem *simples, moderna, de propósito geral e orientada a objetos*. Na época, aproximava-se de Java no estilo; não havia **genéricos**, **LINQ**, **async/await** nem muitos atalhos que usamos hoje — o código tende a ser mais explícito.

Este roteiro não exige compilar com o compilador de 2002: usa **.NET e C# atuais** para experimentar ideias, mantendo o foco no que **C# 1.0 trouxe** como base da linguagem.

## Como usar este material

1. Leia o documento do tópico na ordem acima (cada um prepara o seguinte em parte).
2. Recrie os exemplos num ficheiro `.cs` no projeto console (ou no `Program.cs` temporariamente).
3. Execute `dotnet run` na pasta do projeto `MindSetCSharp1_0.Console`.
4. No fim de cada documento, use as perguntas de reflexão para fixar o “mindset” do tópico.
