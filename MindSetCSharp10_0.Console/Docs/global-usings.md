# Diretivas `global using` (C# 10.0)

## Contexto

**`global using System;`** (e variantes `static` / alias) aplicam-se a **todos** os ficheiros de compilação do projeto — menos repetição e dependências explícitas num único sítio (são frequentemente `GlobalUsings.cs` ou `ImplicitUsings` no SDK).

## Nota

O SDK moderno pode gerar *global usings* implicitamente; aqui o foco é a **funcionalidade da linguagem**.
