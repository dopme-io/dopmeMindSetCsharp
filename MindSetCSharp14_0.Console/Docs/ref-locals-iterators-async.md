# `ref` locals e `unsafe` em iteradores / `async` (C# 13.0)

## Contexto

**`ref` locals** (e contextos **`unsafe`** onde aplicável) passam a ser permitidos em métodos **iteradores** (`yield`) e métodos **`async`**, com as regras de segurança e *lifetime* atualizadas pelo compilador.
