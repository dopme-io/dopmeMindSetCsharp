# `ref struct` como argumento de tipo genérico (C# 13.0)

## Contexto

Pode permitir-se que **`ref struct`** seja usado como **argumento de tipo** para parâmetros de tipo genéricos, quando a restrição / regra da API o explicita — alargando cenários de desempenho com *spans* e tipos semelhantes.

## O que praticar

Consulte a documentação sobre **`allows ref struct`** (restrição em parâmetros de tipo) e exemplos com `Span<T>`.
