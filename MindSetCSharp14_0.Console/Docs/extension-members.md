# *Extension members* (C# 14.0)

## Contexto

Além de métodos de extensão, C# 14 alarga o modelo de **extensões** para outros membros (conforme a especificação e o conjunto suportado pelo compilador) — reduzindo *wrappers* e mantendo estilo fluente sobre tipos existentes.

## O que praticar

Siga exemplos oficiais de **extension properties**, **extension operators** ou membros estáticos de extensão na versão do SDK que utiliza.
