# Escape `\e` (C# 13.0)

## Contexto

Em literais de carácter e *strings*, **`\e`** representa o carácter **ESCAPE** (**Unicode U+001B**), equivalente claro a `\u001b` ou `\x1b`.

## O que praticar

```csharp
char esc = '\e';
```
