# Índice `^` em inicializadores de objeto (C# 13.0)

## Contexto

O operador de índice **“a partir do fim”** **`^`** pode usar-se em **expressões de inicializador de objeto** quando o contexto permite indexação implícita (por exemplo, coleções com indexador exposto no padrão esperado).

## O que praticar

Experimente inicializadores com **`[^1] = …`** em tipos que suportam indexação em inicializadores.
