# C# 3.0 — Roteiro MindSet

**Versão da linguagem:** 3.0  
**Contexto:** **final de 2007**, com o **Visual Studio 2008**; o conjunto completo de funcionalidades alinhou-se sobretudo com o **.NET Framework 3.5**. C# 3.0 consolidou **LINQ** (expressões de consulta + métodos de extensão), **lambdas**, **`var`**, inicializadores, **tipos anónimos**, **árvores de expressão**, **métodos parciais** e **propriedades autoimplementadas** — um passo forte para um estilo **declarativo** e **funcional** em conjunto com OO.

Este roteiro usa o **SDK .NET atual** para exemplos; os textos assinalam o que é próprio de **3.0** e o que são evoluções posteriores.
